Classes used

  • data-toggle : the class of the element which should be effected
  • data-target : the id of the element which should be effected
  • navbar-item for nav items and navbar-link for links
  • navbar-nav to enclose all the nav items
  • navbar-toggler-icon for the hamburger
  • navbar-toggler to disappear hamburger on navbar menu expand
  • navbar-expand-md/lg to keep the menu expanded from that breakpoint
